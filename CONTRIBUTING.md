# 贡献指南

欢迎对这个模板仓库提交改进建议和补丁。为了让模板保持稳定、易维护，提交前请先确认下面几点。

## 提交内容建议

- 优先提交与版式修正、编译兼容性、文档完善相关的改动
- 示例内容保持通用占位文本，不要提交真实学生信息
- 如果改动会影响排版结果，请同步更新文档说明
- 如果改动会影响最终输出，请附上新的 PDF 截图或说明变化位置

## 本地检查

在仓库根目录执行：

```bash
make pdf
```

如果你需要更新仓库自带示例文件，可继续执行：

```bash
make sample
make preview
```

## 提交规范

- 一个提交尽量只做一类事情
- 改类文件时，尽量同时说明影响范围，例如封面、页眉、目录或图表样式
- 不要提交本地编译临时文件，如 `.aux`、`.log`、`.toc`、`main.pdf`
- `docs/sample.pdf` 和 `docs/preview.png` 只有在示例产物确实变化时才更新

## 提 Issue 前

如果是格式问题，请尽量补充：

- 操作系统或 Overleaf 环境
- TeX 发行版
- 编译引擎
- 最小复现示例
- 预期效果和实际效果截图
